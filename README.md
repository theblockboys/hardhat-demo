# Hardhat Demonstration

This project supports the Hardhat Demo videos on The Block Boys YouTube channel.  

Try running some of the following tasks:

```shell
npx hardhat compile
npx hardhat test
npx hardhat node
npx hardhat run --network localhost scripts/deploy.ts
```
